Dashboard Console Debug Output

Open your browser console (F12) and then visit the dashboard in a new tab:

Open Dashboard

Then come back here and paste what you see in the console.

Expected Console Output:

Historical commits data structure: [...]
Extracted metrics: [...]
Skipping broken benchmark for ...
Zero latency for ...

What This Tells Us:

  • If you see "websocket_data: undefined" - the new format isn't being detected
  • If you see "is_broken: true" - commits are being filtered out
  • If you see "latency: 0" - the extraction is failing
  • If you see "latency: null" - no data is available

The console output will reveal exactly why the charts show zeros.
